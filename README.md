# 🦷 SparkDent

 **Live Demo:** [Visit SparkDent](https://sparkdent.netlify.app/)

> A Modern Dental Appointment Booking System
SparkDent is a modern, user-friendly dental clinic web application built using **HTML, CSS, JavaScript, and PHP**.  
The platform allows users to explore dentists, read about their professional background, and book appointments easily with real-time availability validation.

## Admin Panel

SparkDent also includes an admin dashboard where administrators can view all booked appointments, monitor schedules, and manage patient reservations efficiently.

---


## Project Overview

SparkDent is designed to simplify the dental appointment process by providing:

-  Dentist profiles with professional information
-  Smart appointment booking system
-  Real-time time slot validation
-  Clean and modern UI design
-  Responsive layout for all devices

The system prevents double bookings by automatically checking if a selected date and time have already been reserved.

---

## Features

### 1. Dentist Profiles
- View available dentists
- Read about their specialization and experience
- Professional descriptions and service details

### 2. Appointment Booking System
- Select preferred dentist
- Choose date and time
- System verifies availability before confirming booking
- Prevents double booking conflicts

### 3. Smart Validation
- Backend validation using PHP
- Database checks for already reserved time slots
- User-friendly error and success messages

### 4. Modern UI/UX
- Clean and minimal interface
- Smooth interactions using JavaScript
- Fully responsive design
- Professional healthcare aesthetic
- 
### 5. Admin Dashboard
- View all booked appointments
- Monitor patient schedules in real-time
- Manage and organize clinic reservations efficiently
- Prevent scheduling conflicts
